=============================
231. Power of Two
=============================

**Problem Statement**

    Given an integer n, return true if it is a power of two. Otherwise, return false.

    An integer n is a power of two, if there exists an integer x such that n == 2x.

**Example 1:**

    Input: n = 1

    Output: true

    Explanation: 2^0 = 1

**Constraints:**

    * -2^31 <= n <= 2^31 - 1

**Follow up:**
    * Could you solve it without loops/recursion?
